Criar bucket local stack
aws --endpoint-url=http://localhost:4566 s3 mb s3://files-bankslip

Checar se bucket foi criado
aws --endpoint-url=http://localhost:4566 s3 ls

Checar arquivos salvos no bucket
aws --endpoint-url=http://localhost:4566 s3 ls s3://files-bankslip --recursive --human-readable
